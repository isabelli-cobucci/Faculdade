# Funções e atalhos do R

 - `round` -> função que arredonda resultados
 - `ctrl + l` -> limpa console
 - `setas cima e baixo` -> acessa histórico do console
 - `sqrt` -> função que calcula raiz quadrada
 - `mean` -> função que calcula média
 - `rm` -> função que apaga uma variável
 - `median` -> função que calcula mediana
 - `min` -> função que calcula mínimo
 - `max` -> função que calcula máximo
 - `var` -> função que calcula variância
 - `sd` -> função que calcula desvio padrão
